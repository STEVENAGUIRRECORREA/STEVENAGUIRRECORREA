## About me

I'm a Systems Engineer with experience across **QA, process automation, SQL data analysis, software support, and project coordination**.

My work is focused on connecting technical execution with operational impact. I usually participate in initiatives where software, data, business rules, users, and process traceability must work together.

I have worked on automation, reporting, process improvement, technical documentation, database analysis, QA validation, and operational support for business-critical systems.

My profile combines:

- Manual and automated QA.
- SQL analysis and stored procedure optimization.
- Process automation with Power Automate and scripting.
- Data visualization and operational reporting.
- Requirement analysis and user story definition.
- PMO activities such as planning, follow-up, risk tracking, and stakeholder alignment.
- Technical documentation for business and technology teams.
